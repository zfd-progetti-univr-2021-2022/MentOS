◆ realpath()

char* realpath ( const char *  path,
char *  resolved 
)

Return the canonicalized absolute pathname.

Parameters
pathThe path we are canonicalizing.
resolvedThe canonicalized path.
Returns
If there is no error, realpath() returns a pointer to the resolved. Otherwise, it returns NULL, the contents of the array resolved are undefined, and errno is set to indicate the error.

If resolved is NULL, then realpath() uses malloc to allocate a buffer of up to PATH_MAX bytes to hold the resolved pathname, and returns a pointer to this buffer.
