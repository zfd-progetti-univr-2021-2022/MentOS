struct group * getgrent (void)
 Returns a pointer to a structure containing the broken-out fields of an entry in the group database. When first called returns a pointer to a group structure containing the first entry in the group database. Thereafter, it returns a pointer to a group structure containing the next group structure in the group database, so successive calls may be used to search the entire database.
 
void endgrent (void)
 Rewinds the group database to allow repeated searches.
 
void setgrent (void)
 May be called to close the group database when processing is complete.
