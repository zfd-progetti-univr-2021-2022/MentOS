◆ vfs_mount()

int vfs_mount ( const char *  path,
vfs_file_t *  fs_root 
)

Mount a file system to the specified path.

Parameters
pathPath where we want to map the filesystem.
fs_rootRoot node of the filesystem.
Returns
1 on success, 0 on fail.

For example, if we have an EXT2 filesystem with a root node of ext2_root and we want to mount it to /, we would run vfs_mount("/", ext2_root); - or, if we have a procfs node, we could mount that to /dev/procfs. Individual files can also be mounted.
